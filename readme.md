## __Installation__

Create env
python=3.10

```powershell
/path/to/python3.10 -m venv env

or

python -m venv env
```

Install packages
bash

```bash
source env/Scripts/activate

```

```powershell
.\env\Scripts\Activate.ps1
pip install -r requirements.txt
```

Check packages

```powershell
pip list
```
